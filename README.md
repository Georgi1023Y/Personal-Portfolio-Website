# Personal-Portfolio-Website
A modern and interactive personal portfolio website built with React, styled using Tailwind CSS, featuring animations by Framer Motion, 3D elements with Three.js, and backend services powered by Supabase. Showcasing projects, skills, and contact information with a clean and responsive design.

## Overview

This is a personal portfolio website showcasing projects, skills, and contact information. Built using modern web technologies including **React**, **Tailwind CSS**, **Framer Motion**, **Three.js**, and **Supabase**, it features a clean and interactive design with 3D elements and animations.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Framer Motion**: A library for animations and transitions.
- **Three.js**: A JavaScript library for creating 3D graphics.
- **Supabase**: An open-source backend-as-a-service for managing data and authentication.

## Features

- Interactive 3D model integrated with Three.js
- Responsive design with Tailwind CSS
- Smooth animations powered by Framer Motion
- Backend services for data and authentication with Supabase
